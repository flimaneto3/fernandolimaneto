# Fernando Neto Consulting - Static Website

Simple one-page static website for a technology consultancy focused on .NET development, Azure cloud solutions and AI tools.

## Files

- `index.html` - page content and structure
- `styles.css` - visual design and responsive layout
- `script.js` - mobile menu and dynamic year

## How to use

Open `index.html` in a browser.

Before publishing, replace:

- `contact@example.com` with your real email
- `FN Consulting` with your final brand/company name if needed
- Any text you want to personalize further

## Deployment options

This can be deployed to GitHub Pages, Azure Static Web Apps, Netlify, Vercel or any static hosting provider.
